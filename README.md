# Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## Project Overview

This project applies Machine Learning techniques to analyze buyer behavior in the real estate market. The objective is to identify distinct customer segments using K-Means Clustering and generate actionable investment profiles for business intelligence.

## Dataset Information

### Client Dataset

* Records: 2,000
* Features: 12

### Property Dataset

* Records: 10,000
* Features: 9

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Google Colab

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Label Encoding
5. Feature Scaling
6. K-Means Clustering
7. Elbow Method
8. Silhouette Score Evaluation
9. Buyer Profiling

## Results

### Optimal Clusters

* K = 5

### Silhouette Score

* 0.163

### Buyer Segments

| Cluster | Segment                    |
| ------- | -------------------------- |
| 0       | International Investors    |
| 1       | Investment-Oriented Buyers |
| 2       | Loan-Dependent Home Buyers |
| 3       | Self-Financed Home Buyers  |
| 4       | Corporate Investors        |

## Project Outputs

* Clustered Buyer Dataset
* Cluster Summary Report
* Buyer Profile Report

* Market Intelligence Insights
